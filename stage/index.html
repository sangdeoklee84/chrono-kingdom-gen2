설치 · 계정

  • Higgsfield 계정 — 가입·로그인, 이미지 생성은 크레딧/유료 필요할 수 있음

저장소 접근 권한

  • chrono-stage는 비공개 저장소
  • 관리자에게 협업자 초대를 받고 수락
  • git이 GitHub에 인증돼 있어야 클론됨 (HTTPS면 Personal Access Token)

⚠️ Windows에서 스크립트가 막히면(running scripts is disabled): powershell -ExecutionPolicy Bypass -File install.ps1

빠른 시작 — 1분 설치

두 방법 모두 아래 2단계로 준비됩니다. 설치 스크립트는 시스템을 바꾸지 않는 안전한 점검형(파일 복사·안내만)이라 부담 없이 실행하세요.

1

저장소 클론

안에 README.md(사용법)와 STAGE-BACKGROUNDS.md(화풍·구도 템플릿)가 들어 있습니다.

git clone https://github.com/sangdeoklee84/chrono-stage.git
cd chrono-stage
2
한 번만 실행

설치 스크립트 실행

필수 도구 점검 → .env 자동 생성 → 다음 단계 안내까지 한 번에 끝납니다. OS에 맞는 것만 실행하세요.

# Windows  (탐색기에서 install.ps1 우클릭 → "PowerShell로 실행" 도 가능)
./install.ps1

# macOS / Linux
bash install.sh

스크립트가 끝나면 화면 안내대로 방법 A(권장) 또는 방법 B로 진행합니다. ↓

방법 A · Higgsfield

권장 · 설치 불필요

자기 Higgsfield 계정을 Claude Code에 연결합니다. 기본 방법(.mcp.json)을 우선 쓰고, 안 되면 보조 방법.

1
기본 · .mcp.json (http)

프로젝트 MCP로 연결

클론한 chrono-stage 폴더엔 .mcp.json이 있어, 이 폴더에서 Claude Code를 열면 higgsfield 서버가 자동 등록됩니다(허용 프롬프트 승인). 그다음 /mcp → higgsfield → 브라우저 로그인 + Authorize → 다시 /mcp로 connected 확인.

+
추가 · claude.ai 관리형 커넥터

대안 연결

기본이 안 되거나 claude.ai 커넥터를 선호하면: /mcp → "claude.ai Higgsfield" 선택 → 로그인 + Authorize. 둘 중 하나만 쓰세요(동시 사용 시 도구 중복).

2

이미지 요청 — 그냥 말로

초원 드루이드 유적 스테이지 배경을 GPT Image 2 16:9로 만들어줘
3

마무리 — 투명 / 규격화

스테이지는 tools/crop-2048x1024.ps1로 2048×1024 규격화. 배경 제거는 하지 않습니다(꽉 찬 배경).

방법 B · Gemini / GPT 스크립트 (Higgsfield 없이)

Higgsfield 계정이 없어도 OK — Gemini 또는 GPT(OpenAI) API 키만으로 생성. Node 20.6+ 필요(--env-file).

1
최초 1회만

키 채우기 (.env)

설치 스크립트가 만든 .env에 쓰는 쪽 키만 넣으면 됩니다. (둘 다 결제 필요)

# 둘 중 쓰는 것만:
GEMINI_API_KEY=본인_키
OPENAI_API_KEY=본인_키
2

Gemini로 생성 / 편집

node --env-file=.env gemini/gen.mjs "프롬프트" out/결과.png
# 기존 이미지 편집: ... --in 원본.png
3

GPT(OpenAI)로 생성

node --env-file=.env openai/gen.mjs "프롬프트" out/결과.png --size 1536x1024

size: 1024x1024 / 1536x1024 / 1024x1536. 스테이지는 이후 crop-resize로 규격화.

⚠️ 이 경로는 투명배경 자동 컷아웃이 없음 — 마을 에셋은 단색 배경 생성 후 외부 도구로 컷아웃 필요(Higgsfield 경로는 자동).

🌐 API·CLI 없이? 웹에서 수동

키도 Claude Code도 없으면, 브라우저 웹 UI(ChatGPT / Gemini / Higgsfield)만으로도 됩니다.

💬 ChatGPT만 쓰는 팀원용 (무설치 요약)

  1. 아래 복붙 예시를 붙여넣고 가로(Landscape)로 생성 — ChatGPT엔 16:9·2:1 옵션이 없어 1536×1024로 나옵니다.
  2. 3종 화풍 통일: 첫 결과 이미지를 같은 대화에 첨부하고 "같은 스타일로 오브젝트만 교체"라고 요청.
  3. 규격화(2048×1024): Photopea 무료 웹(설치·가입 불필요)에서 2048×1024 캔버스에 맞춰 PNG로 내보내기.
  4. 스테이지는 배경 제거 안 함(꽉 찬 배경).

순서

  1. 웹 서비스 로그인 (ChatGPT 권장 / Gemini / Higgsfield)
  2. 아래 "복붙 예시" 프롬프트 복사 → 붙여넣고 가로(Landscape)로 생성
  3. 결과 다운로드 (ChatGPT는 16:9·2:1 옵션이 없어 1536×1024로 나옴)
  4. 규격화(무설치) — Photopea 무료 웹: File→New 2048×1024 → 이미지 드래그 → 꽉 맞춤 → File→Export as→PNG
  5. 배경 제거 안 함 (꽉 찬 배경)

바로 복붙 예시 (스테이지)

밝고 깔끔한 카툰 2D 스테이지 배경, 외곽선 없음. 벨트스크롤·좌우 루프(양 끝이 오브젝트를 가로질러 반복), 가운데 집중·중앙 히어로 금지. 큼직한 덩어리 데포르메. 모든 오브젝트 밑동은 화면 약 48% 한 지면선에 정렬, 밀집은 상단 20~48%, 50% 아래로는 아무것도 없음(하단 절반은 빈 모래 플레이 레인, 긴 그림자만). 전경은 거의 비우고 큰 전경물(말뚝·유목·덤불) 금지 — 맨 아래 모서리 작은 잘린 실루엣만. 좌상단 단일 광원, 파스텔·낮은 콘트라스트(검정/순백/형광 금지). [바이옴: 화산]. 실사·3D·CGI 금지.

⚠️ 화풍·규격을 사람이 직접 챙겨야 함. 반복·대량 작업엔 방법 A~B 권장.

🗺️ ② 필드 스테이지 만들기 (페인터리)

횡스크롤 전투 배경. 레퍼런스 실측 기반 규정. 상세는 STAGE-BACKGROUNDS.md.

화풍 · 형태

  • 밝고 깔끔한 카툰 일러스트 — 실사·3D 금지
  • 큼직한 덩어리 데포르메
  • 청록/녹색 룬 글리프 모티프
  • 일관된 단일 광원

구도 · 배치

  • 일반 스테이지 = 벨트스크롤 (가운데 집중 ❌)
  • 리듬감 — 크고작은 + 공백 (균등반복 ❌)
  • 좌우 타일 / 루프
  • 오브젝트 상단 절반, 하단 절반 빈 레인 (50%)
  • 유닛 레인 밝고 단순 — 캐릭터 가독성
  • HUD 세이프존 — 정중앙 16:9만 보임, 상단 모서리·하단 중앙 조용히

색 · 톤

  • 파스텔 · 낮은 콘트라스트
  • 팔레트 양 끝단 금지 (검정/순백/형광 ❌)
  • 색 다양성 유지 (베이스 + 포인트·보색)
  • 전경은 거의 비움 — 큰 전경물 ❌, 깊이는 바닥 그림자로

만드는 순서 · 규격

  1. 요청 — "화산지대 스테이지 만들어줘"
  2. 16:9로 생성 (GPT Image 2 권장)
  3. 2048×1024로 크롭 — crop-2048x1024.ps1 (보스 2600×1600·투기장 2560×1080은 crop-resize)
  4. 오브젝트 디테일만 바꿔 변형
  5. (선택) style-anchors/stage_stylesheet.jpg(중립 6바이옴 몽타주) 첨부로 공식 화풍에 더 똑같이 (image-to-image)

최신 규정 + 중립앵커(stage_stylesheet) 적용 예시 — 항구 3종 (채택본)

⛔ 절대 하지 말 것 — 보안

  • API 키를 코드·문서·git에 넣지 않기. 반드시 .env 또는 환경변수로만.
  • .env는 이미 .gitignore 처리됨 — 커밋되지 않습니다.
  • 생성 결과물 out/도 git에 올리지 않습니다. 각자 로컬에서 생성.
  • 키가 노출되면 즉시 폐기 후 재발급 (Google AI Studio).
