0단계 · 사전 준비 (처음 한 번)

아래가 없으면 시작할 수 없습니다. 한 번만 갖추면 됩니다.

설치 · 계정

  • Claude Code 설치 + 로그인 — npm install -g @anthropic-ai/claude-code (또는 데스크톱 앱). Anthropic 구독/결제 필요
  • git 설치 — 클론용, 클론 전에 먼저
  • Node.js 20.6+ — 방법 B/C·도구용 (--env-file)
  • Higgsfield 계정 — 가입·로그인, 이미지 생성은 크레딧/유료 필요할 수 있음

저장소 접근 권한

  • chrono-village는 비공개 저장소
  • 관리자에게 협업자 초대를 받고 수락
  • git이 GitHub에 인증돼 있어야 클론됨 (HTTPS면 Personal Access Token)

⚠️ Windows에서 스크립트가 막히면(running scripts is disabled): powershell -ExecutionPolicy Bypass -File install.ps1

빠른 시작 — 1분 설치

두 방법 모두 아래 2단계로 준비됩니다. 설치 스크립트는 시스템을 바꾸지 않는 안전한 점검형(파일 복사·안내만)이라 부담 없이 실행하세요.

1

저장소 클론

안에 README.md(사용법)와 PROMPTING.md(화풍 템플릿)가 들어 있습니다.

git clone https://github.com/sangdeoklee84/chrono-village.git
cd chrono-village
2
한 번만 실행

설치 스크립트 실행

필수 도구 점검 → .env 자동 생성 → 다음 단계 안내까지 한 번에 끝납니다. OS에 맞는 것만 실행하세요.

# Windows  (탐색기에서 install.ps1 우클릭 → "PowerShell로 실행" 도 가능)
./install.ps1

# macOS / Linux
bash install.sh

스크립트가 끝나면 화면 안내대로 방법 A(권장) 또는 방법 B로 진행합니다. ↓

방법 A · Higgsfield

권장 · 설치 불필요

자기 Higgsfield 계정을 Claude Code에 연결합니다. 기본 방법(.mcp.json)을 우선 쓰고, 안 되면 보조 방법.

1
기본 · .mcp.json (http)

프로젝트 MCP로 연결

클론한 chrono-village 폴더엔 .mcp.json이 있어, 이 폴더에서 Claude Code를 열면 higgsfield 서버가 자동 등록됩니다(허용 프롬프트 승인). 그다음 /mcp → higgsfield → 브라우저 로그인 + Authorize → 다시 /mcp로 connected 확인.

+
추가 · claude.ai 관리형 커넥터

대안 연결

기본이 안 되거나 claude.ai 커넥터를 선호하면: /mcp → "claude.ai Higgsfield" 선택 → 로그인 + Authorize. 둘 중 하나만 쓰세요(동시 사용 시 도구 중복).

2

이미지 요청 — 그냥 말로

마녀모자 쓴 할로윈 주점을 GPT Image 2로 만들어줘
3

마무리 — 투명 / 규격화

마을 에셋은 "배경 투명하게"(remove_background)로 투명 PNG 컷아웃하면 끝.

방법 B · Gemini / GPT 스크립트 (Higgsfield 없이)

Higgsfield 계정이 없어도 OK — Gemini 또는 GPT(OpenAI) API 키만으로 생성. Node 20.6+ 필요(--env-file).

1
최초 1회만

키 채우기 (.env)

설치 스크립트가 만든 .env에 쓰는 쪽 키만 넣으면 됩니다. (둘 다 결제 필요)

# 둘 중 쓰는 것만:
GEMINI_API_KEY=본인_키
OPENAI_API_KEY=본인_키
2

Gemini로 생성 / 편집

node --env-file=.env gemini/gen.mjs "프롬프트" out/결과.png
# 기존 이미지 편집: ... --in 원본.png
3

GPT(OpenAI)로 생성

node --env-file=.env openai/gen.mjs "프롬프트" out/결과.png --size 1536x1024

size: 1024x1024 / 1536x1024 / 1024x1536. 스테이지는 이후 crop-resize로 규격화.

⚠️ 이 경로는 투명배경 자동 컷아웃이 없음 — 마을 에셋은 단색 배경 생성 후 외부 도구로 컷아웃 필요(Higgsfield 경로는 자동).

🌐 API·CLI 없이? 웹에서 수동

키도 Claude Code도 없으면, 브라우저 웹 UI(ChatGPT / Gemini / Higgsfield)만으로도 됩니다.

순서

  1. 웹 서비스 로그인 (ChatGPT / Gemini / Higgsfield)
  2. 프롬프트 템플릿 복사 (PROMPTING.md)
  3. 붙여넣고 생성 → 다운로드
  4. 색·시즌 변형으로 세트화
  5. (마을·투명 필요) remove.bg 등으로 배경 제거

바로 복붙 예시 (마을)

둥근 셀셰이딩 2D 게임 오브젝트, 짙은 갈색 외곽선(#3A2A1E, 검정 금지), 2~3단 셀셰이딩, 밝고 부드러운 저채도 파스텔(코랄 포인트), 얇은 짙은 갈색 외곽선, 30~45도 쿼터뷰, 투명 배경+드롭섀도우. 2:1 아이소 다이아몬드 타일(윗면 약 27도) 위에 바닥 접지면만 타일 안. [대상: 빵집]. 어둡거나 고채도 금지, 실사 금지.

⚠️ 화풍·규격을 사람이 직접 챙겨야 함. 반복·대량 작업엔 방법 A~B 권장.

🏘️ ① 마을 에셋 만들기 (셀셰이딩)

마을에 배치할 건물·소품. 상세 프롬프트는 저장소 PROMPTING.md.

화풍 규칙

  • 외곽선 짙은 갈색 #3A2A1E — 얇고 부드럽게 (검정·니어블랙 ❌)
  • 2~3단 셀셰이딩 — 그라데이션 ❌, 면 분리
  • 밝고 부드러운 저채도 파스텔(코랄 오렌지는 포인트만) — 공식 건물 톤, 어둡거나 고채도 ❌
  • 둥글고 통통, 30~45° 쿼터뷰
  • 투명 배경(알파) + 바닥 드롭섀도우
  • 베이스 타일 — 2:1 아이소 다이아몬드(윗면 ~27°, 바닥_03 기준), 바닥 접지 footprint만 타일 안 (지붕·모자 등 돌출부는 밖으로 OK)

만드는 순서

  1. Claude에게 요청 — "둥근 셀셰이딩 빵집 만들어줘"
  2. 결과 확인 (GPT Image 2 / Nano Banana)
  3. "배경 투명하게" → 컷아웃(remove_background)
  4. 색·시즌 변형으로 세트화
  5. (선택) style-anchors/village_stylesheet.jpg 첨부로 화풍·베이스타일 통일

예시: 마녀모자 할로윈 주점 (투명 PNG)

⛔ 절대 하지 말 것 — 보안

  • API 키를 코드·문서·git에 넣지 않기. 반드시 .env 또는 환경변수로만.
  • .env는 이미 .gitignore 처리됨 — 커밋되지 않습니다.
  • 생성 결과물 out/도 git에 올리지 않습니다. 각자 로컬에서 생성.
  • 키가 노출되면 즉시 폐기 후 재발급 (Google AI Studio).
